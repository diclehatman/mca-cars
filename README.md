# 🚗 MCA CARS — İkinci El Araç Alım Satım Web Sitesi

> 🎓 Bu proje, üniversitenin **1. sınıfında** geliştirdiğim **ilk web sitesidir.**  
> Sıfırdan HTML, CSS ve JavaScript öğrenerek tasarladım.

---

## 📌 Proje Hakkında

**MCA CARS**, İstanbul Bağcılar'da faaliyet gösteren ikinci el araç alım satım galerisinin tanıtım ve iletişim web sitesidir. Müşterilere güvenilir, şeffaf ve hızlı araç alım satım deneyimi sunmayı hedefler.

---

## 🌐 Sayfalar

| Sayfa | Dosya | Açıklama |
|---|---|---|
| Ana Sayfa | `index.html` | Video arka planlı hero bölümü, Sahibinden.com yönlendirmesi |
| Hakkımızda | `hakkimizda.html` | Şirket tanıtımı, güven ve kalite bölümleri, özellik kartları |
| Şubemiz | `subemiz.html` | Google Haritalar embed, yol tarifi butonu |
| İletişim | `iletisim.html` | İletişim formu (Formspree entegrasyonu) |

---

## 🛠️ Kullanılan Teknolojiler

- **HTML5** — Sayfa yapısı
- **CSS3** — Tasarım ve animasyonlar (`global.css`)
- **JavaScript** — Mobil menü toggle, scroll efekti
- **Font Awesome 6** — İkonlar
- **Google Fonts** — Inter yazı tipi
- **Formspree** — İletişim formu arka ucu
- **Google Maps Embed API** — Şube haritası

---

## 📁 Dosya Yapısı

```
Mca-main/
├── index.html          # Ana sayfa
├── hakkimizda.html     # Hakkımızda sayfası
├── subemiz.html        # Şube ve harita sayfası
├── iletisim.html       # İletişim formu sayfası
├── global.css          # Ana stil dosyası (tüm sayfalar)
├── script.js           # JavaScript (menü, scroll efekti)
└── assets/     # Logo, görseller ve arka plan videosu
```

---

## ✨ Özellikler

- 📱 **Responsive tasarım** — Mobil ve masaüstü uyumlu
- 🎬 **Video arka plan** — Ana sayfada otomatik döngülü video
- 🧭 **Sabit navbar** — Scroll ile glassmorphism efekti
- 📨 **İletişim formu** — Formspree ile e-posta gönderimi
- 🗺️ **Google Haritalar** — Şube konumu embedded harita
- 🎨 **Animasyonlar** — Hover efektleri, fade-in geçişleri

## 🚀 Nasıl Çalıştırılır?

Herhangi bir sunucu ya da kurulum gerekmez.  
`index.html` dosyasını doğrudan **Safari**, **Chrome** veya herhangi bir tarayıcıda açabilirsiniz.

```bash
# macOS'ta terminalle açmak için:
open -a Safari index.html
```
